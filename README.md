# Excel Correlation Heatmap: Supply Chain Analytics

This repository contains a correlation analysis of supply chain metrics using Excel-style heatmap visualization.

## Files

- **correlation.csv** - Correlation matrix values showing relationships between supply chain variables
- **heatmap.png** - Heatmap visualization with Red-White-Green conditional formatting
- **README.md** - Project documentation
